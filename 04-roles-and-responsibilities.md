# Roles & Responsibilities

## Tech Guy
- Onboard new clients end-to-end
- Build all ad creative in CapCut
- Launch ads on Meta
- Install Meta pixel
- Configure GHL sub-accounts (pipelines, automations, calendars, forms)
- Hand off to CSMs once client is live

## CSM (Customer Success Manager / Setter)
- Call all leads coming in for assigned clients during their shift
- Book appointments into the client's calendar
- Be the primary point of contact for assigned clients in Slack
- Answer client questions, escalate to me only when needed
- Update each client's "Current Happiness" rating (1–10) in the dashboard
- Leave notes per client in the dashboard
- Hit appointment-generation targets

## Me
- Run B2B ads for Rambitious Media (my own acquisition)
- Sales calls + closing new PPSA clients
- Manage tech guy + CSMs
- Assign clients to CSMs via Rambitious OS
- Review pod health weekly (happiness scores, appt volume, client churn risk)
- Strategy + offer evolution
